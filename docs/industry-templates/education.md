# Industry Template: Education

## Goal

Convert course/program inquiries into qualified admissions conversations.

## Funnel

1. Program page.
2. WhatsApp tracked CTA.
3. Student/parent qualification.
4. Admissions handoff.
5. Enrollment follow-up.

## WhatsApp Sequence

- First reply: ask desired program and level.
- Follow-up 1: explain next intake and requirements.
- Follow-up 2: invite call or campus/online consultation.
- Follow-up 3: deadline reminder without false urgency.

## Qualification Rules

- Program interest.
- Student level.
- Start date.
- Budget/payment preference.
- Parent/decision maker where relevant.
